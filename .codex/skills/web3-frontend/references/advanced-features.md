# PRD Expansion (MVP+)

## NFT Certificates
- Trigger: course completion
- UI: "Get Certificate" button on course completion
- Display NFT badge on course detail and profile
- Link to block explorer for certificate token

## Learning Progress
- Track progress per course (chapter/percent)
- Display progress bar on course card and detail
- Sync progress to backend (and optionally onchain hash)

## Admin Console
- Admin-only view for course moderation
- Actions: approve, unpublish, freeze course
- Table view for users/courses/transactions

## Analytics
- Dashboard with key metrics:
  - DAU/WAU
  - Purchase conversion
  - Completion rate
  - Revenue
- Basic chart components + filter by date
